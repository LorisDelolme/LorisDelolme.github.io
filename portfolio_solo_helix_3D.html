Helix 3D

A 3D version of Kuru kuru kururin

Concept

This is the 3D version of a previous prototype. The goal was to play with the camera placement to add new gameplay dimensions.
The prototype offers a sandbox template like the 2d version but do not use splines to creates the levels and only use simple blocking system (made with Umodeler).
The goal of this prototype was to transform 2D scripts into 3d ones in one week end. The 2d version offers more tools to create levels but 3D camera angles allows the creation of varieted moments.

Collectibles

It is possible to collect bonus and malus to interact with the player ship. As seen in the gif, a bonus can inverse the rotation speed or rotation way.

Creating enemies and obstacles

There are 3 types of enemies in this prototype. The first is a rotating board mimicking the player. The second one is moving cube (but can also be static), moving between severals points of a path. The last one is a projectiles spawner creating balls to avoid that can be used to create alternating patterns in a corridor.
